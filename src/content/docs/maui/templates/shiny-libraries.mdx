---
title: Shiny Libraries
---

The Shiny Templates come pre-configured with the following Shiny libraries. Each is AOT-compatible and built with source generators for zero-reflection performance.

## Core

| Library | Description |
|---|---|
| <a href="/client/appbuilder/" target="_blank">Shiny.NET</a> | Core framework for building mobile applications with background services, BLE, jobs, GPS, notifications, and more |
| <a href="/mediator/" target="_blank">Shiny Mediator</a> | Mediator pattern implementation with middleware, caching, offline support, and AOT source generation |
| <a href="/maui/shell/" target="_blank">Shiny MAUI Shell</a> | Shell navigation with ViewModel lifecycle management, source-generated routes, and DI integration |

## Extensions

| Library | Description |
|---|---|
| <a href="/extensions/di/" target="_blank">Shiny Extensions DI</a> | Attribute-driven, source-generated service registrations with category-based conditional registration |
| <a href="/extensions/stores/" target="_blank">Shiny Stores</a> | Cross-platform key/value store abstraction with persistent service binding |
| <a href="/extensions/reflector/" target="_blank">Shiny Reflector</a> | Source-generated property access, JSON serialization, and assembly info generation |
| <a href="/extensions/localizegen/" target="_blank">Shiny Localization Generator</a> | Strongly-typed localization classes generated from .resx files |
| <a href="/extensions/webhost/" target="_blank">Shiny Web Hosting</a> | ASP.NET Core infrastructure modules |

## Data

| Library | Description |
|---|---|
| <a href="/data/sqlite-docdb/" target="_blank">Shiny SQLite Document DB</a> | Document-oriented SQLite database with LINQ querying, projections, and AOT support |
| <a href="/data/spatial/" target="_blank">Shiny Spatial</a> | Dependency-free spatial database using SQLite R*Tree indexing with custom geometry algorithms |

## MAUI

| Library | Description |
|---|---|
| <a href="/maui/tableview/" target="_blank">Shiny TableView</a> | Settings-style TableView with 14 cell types, cascading styles, sections, and full MVVM support |
