A .NET MAUI controls library providing rich, ready-to-use UI components for mobile and desktop applications. One package covers TableView, Scheduler, BottomSheet, Fab/FabMenu, PillView, SecurityPin, and ImageViewer. Markdown and Mermaid Diagrams are available as separate add-on packages.

## Controls

| Control | Description |
|---|---|
| [**TableView**](/maui/tableview/) | Settings-style table with 14 cell types, cascading styles, drag-to-sort, sections, and full MVVM support |
| [**Scheduler**](/maui/scheduler/) | Calendar grid, agenda timeline, and event list views with a shared `ISchedulerEventProvider` interface |
| [**BottomSheet**](/maui/bottomsheet/) | Draggable bottom sheet overlay with configurable snap points (detents), backdrop, and keyboard handling |
| [**Fab & FabMenu**](/maui/fab/) | Material-style floating action button and expanding multi-action menu with staggered animations, backdrop, and full color customization |
| [**PillView**](/maui/pillview/) | Status badge/label with 6 preset themes, custom colors, and WCAG-accessible contrast |
| [**SecurityPin**](/maui/securitypin/) | PIN entry with individually rendered cells, optional character masking, and full styling control |
| [**ImageViewer**](/maui/imageviewer/) | Full-screen image overlay with pinch-to-zoom, pan, double-tap zoom, and animated transitions |
| [**Markdown**](/maui/markdown/) | Read-only markdown renderer and full editor with formatting toolbar and live preview (separate package) |
| [**Mermaid Diagrams**](/maui/mermaid-diagrams/) | Native Mermaid flowchart rendering with Sugiyama layout, themes, and pan/zoom (separate package) |

## Setup

<LibBuilder componentName="tableview" client:load />

Add the XAML namespace to your pages:

```xml
xmlns:shiny="http://shiny.net/maui/controls"
```

For Markdown controls (separate package: `Shiny.Maui.Controls.Markdown`):

```xml
xmlns:md="http://shiny.net/maui/markdown"
```

For Mermaid Diagrams (separate package: `Shiny.Maui.Controls.MermaidDiagrams`):

```xml
xmlns:diagram="http://shiny.net/maui/diagrams"
```
