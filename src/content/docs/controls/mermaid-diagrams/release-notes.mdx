## v1.0 - TBD

<RN type="feature">**MermaidDiagramControl** — MAUI `ContentView` that parses Mermaid flowchart syntax and renders diagrams natively using MAUI Graphics</RN>
<RN type="feature">**Mermaid parser** — Hand-written recursive-descent parser supporting `graph`/`flowchart` keywords with `TD`, `LR`, `BT`, `RL` directions</RN>
<RN type="feature">**Node shapes** — Rectangle, rounded rectangle, stadium `([text])`, circle `((text))`, diamond `{text}`, and hexagon `{{text}}`</RN>
<RN type="feature">**Edge styles** — Solid `-->`, dotted `-.->`, and thick `==>` lines with arrow or open endings and optional `|label|` syntax</RN>
<RN type="feature">**Edge chaining** — Parser supports `A --> B --> C` chain syntax creating multiple edges in a single statement</RN>
<RN type="feature">**Subgraph support** — Nested `subgraph`/`end` blocks with titles and bracket syntax `subgraph id [Title]`</RN>
<RN type="feature">**Sugiyama layout engine** — Automatic layered graph layout with cycle removal, layer assignment, crossing minimization, node positioning, and edge routing</RN>
<RN type="feature">**4 built-in themes** — Default (blue/white), Dark (navy/red), Forest (green), and Neutral (gray) themes with full customization via `DiagramTheme` subclassing</RN>
<RN type="feature">**Per-node styling** — Override `GetNodeStyle(int index)` on a theme to vary node colors by position</RN>
<RN type="feature">**Gesture support** — Pan and pinch-to-zoom gestures with `AllowPanning` and `AllowZooming` toggle properties</RN>
<RN type="feature">**DiagramLayoutOptions** — Configurable node width/height, horizontal/vertical spacing, subgraph padding, font size, and margin</RN>
<RN type="feature">**Parse error handling** — `ParseError` bindable property exposes error messages for live editor validation</RN>
<RN type="feature">**Programmatic API** — `MermaidParser.Parse()` and `SugiyamaLayoutEngine.Layout()` available independently of the control for headless use</RN>
<RN type="feature">**AOT-safe** — Fully compatible with .NET trimming and NativeAOT</RN>
