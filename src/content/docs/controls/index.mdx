A rich, ready-to-use UI controls library for both .NET MAUI and Blazor. One package per host covers TableView, Scheduler, FloatingPanel, Fab/FabMenu, PillView, SecurityPin, SignaturePad, ImageViewer, ImageEditor, ChatView, ColorPicker, FontPicker, AutoCompleteEntry, CountryPicker, AddressEntry, and TextToSpeechButton. Markdown and Mermaid Diagrams are available as separate add-on packages for each host.

## Controls

| Control | Description |
|---|---|
| [**TableView**](/controls/tableview/) | Settings-style table with 14 cell types, cascading styles, drag-to-sort, sections, and full MVVM support |
| [**Scheduler**](/controls/scheduler/) | Calendar grid, agenda timeline, and event list views with a shared `ISchedulerEventProvider` interface |
| [**FloatingPanel**](/controls/floatingpanel/) | Draggable floating panel overlay (bottom, bottom with tabs, or top) with configurable detents, OverlayHost backdrop management, header peek, keyboard handling, and ShinyContentPage convenience base class |
| [**Fab & FabMenu**](/controls/fab/) | Material-style floating action button and expanding multi-action menu with staggered animations, backdrop, and full color customization |
| [**PillView**](/controls/pillview/) | Status badge/label with 6 preset themes, custom colors, and WCAG-accessible contrast |
| [**SecurityPin**](/controls/securitypin/) | PIN entry with individually rendered cells, optional character masking, and full styling control |
| [**ImageViewer**](/controls/imageviewer/) | Full-screen image overlay with pinch-to-zoom, pan, double-tap zoom, and animated transitions |
| [**ImageEditor**](/controls/imageeditor/) | Inline image editor with crop, rotate, draw, line/arrow, text annotations with font selection, undo/redo, and export to PNG/JPEG/WEBP |
| [**FontPicker**](/controls/fontpicker/) | Font family and font size picker controls with inline list and popup button variants, each font rendered in its own typeface |
| [**ChatView**](/controls/chatview/) | Modern chat UI with message bubbles, typing indicators, per-participant colors/avatars, load-more, and input bar |
| [**ColorPicker**](/controls/colorpicker/) | Full-featured color picker with spectrum view, hue bar, opacity slider, hex input, and preview swatch |
| [**AutoCompleteEntry**](/controls/autocomplete/) | Text input with debounced search, dropdown suggestions, busy indicator, and custom item templates |
| [**CountryPicker**](/controls/countrypicker/) | Country search with flag emoji, name, and dial code from full ISO 3166-1 list |
| [**AddressEntry**](/controls/addressentry/) | Address search with geocoding (Nominatim by default) and structured address results |
| [**SignaturePad**](/controls/signaturepad/) | Signature capture with canvas drawing, PNG export, and configurable panel overlay (requires OverlayHost or ShinyContentPage on MAUI) |
| [**TextToSpeechButton**](/controls/texttospeech/) | Button that speaks bound text using native TTS, with cancel/resume, pitch/volume/locale options, and haptic feedback |
| [**Markdown**](/controls/markdown/) | Read-only markdown renderer and full editor with formatting toolbar and live preview (separate package) |
| [**Mermaid Diagrams**](/controls/mermaid-diagrams/) | Native Mermaid flowchart rendering with Sugiyama layout, themes, and pan/zoom (separate package) |

## Setup

<LibBuilder componentName="tableview" client:load />

### .NET MAUI

Add the XAML namespace to your pages:

```xml
xmlns:shiny="http://shiny.net/maui/controls"
```

For Markdown controls (separate package: `Shiny.Maui.Controls.Markdown`):

```xml
xmlns:md="http://shiny.net/maui/markdown"
```

For Mermaid Diagrams (separate package: `Shiny.Maui.Controls.MermaidDiagrams`):

```xml
xmlns:diagram="http://shiny.net/maui/diagrams"
```

### Blazor

Install `Shiny.Blazor.Controls` (plus `Shiny.Blazor.Controls.Markdown` or `Shiny.Blazor.Controls.MermaidDiagrams` as needed) and add the `@using` directives — typically in `_Imports.razor`:

```razor
@using Shiny.Blazor.Controls
@using Shiny.Blazor.Controls.Cells
@using Shiny.Blazor.Controls.Sections
@using Shiny.Blazor.Controls.Scheduler
@using Shiny.Blazor.Controls.Chat
@using Shiny.Blazor.Controls.Markdown
@using Shiny.Blazor.Controls.MermaidDiagrams
```

No DI registration is required — drop the components into any `.razor` page.
