Sentry provides great ways of tracking performance across transactions and spans within those transaction.  As such, it provides
a great way of tracking performance of your mediator calls.

[Sentry Website](https://sentry.io/welcome)

:::note
This extension provides tracking across ALL mediator message types - Commands, Requests, Streams, and Events.  It also includes
an exception handler that logs and enriches those logs with additional information such as the message and headers.
:::

<Steps>
1. Install the <NugetBadge name="Shiny.Mediator.Sentry" showLabel={true} /> package
2. Call the hosting UseSentry (ie. MauiProgram.cs - UseSentry with your DSN)
3. Register in your against your hosted services call
    ```csharp
    services.AddShinyMediator(cfg =>
    {
        cfg.UseSentry();
    });
    ```
4. Start making calls to the mediator as your normally would
</Steps>
