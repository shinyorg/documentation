Extension often go beyond standard middleware by integrating with 3rd party components on different levels.

| Extension | Description |
|-----------|-------------|
| [MAUI](./maui) | .NET MAUI support for shell navigation, view event collection, & more|
| [Uno Platfor](./blazor.mdx) | Uno Platform support for offline, caching, & more |
| [Blazor](./blazor.mdx) | Blazor support for view event collectors, offline, caching, & more |
| [ASP.NET](./aspnet.mdx) | Your request & command handlers meet minimal routing |
| [Prism](./prism) | Strongly Typed Prism Navigation for .NET MAUI |
| [HTTP](./http) | Strongly Typed HTTP Requests without the boilerplate |
| [Dapper](./dapper) | Dapper support for request handlers |
