## 1.0 - March 6, 2026
<RN type="feature">Initial Public Release</RN>
<RN type="feature">Two-pass query pipeline with R*Tree bounding box filter (SQL) followed by C# geometry refinement</RN>
<RN type="feature">Dual coordinate systems: WGS84 (Haversine distance) and Cartesian (Euclidean distance)</RN>
<RN type="feature">Full geometry type support: Point, LineString, Polygon, MultiPoint, MultiLineString, MultiPolygon, GeometryCollection</RN>
<RN type="feature">Well-Known Binary (WKB) serialization for all geometry types</RN>
<RN type="feature">Fluent query builder with spatial filters, property filters, sorting, and paging</RN>
<RN type="feature">Bulk insert with transaction-wrapped batch operations</RN>
<RN type="feature">Pre-built databases: US states, US cities, Canadian provinces, and Canadian cities</RN>
<RN type="feature">Direct query methods: FindInEnvelope, FindIntersecting, FindWithinDistance, FindNearestNeighbors</RN>
<RN type="feature">AOT-compatible and trimmable with zero geospatial dependencies</RN>
<RN type="feature">Shiny.Spatial.Geofencing bringing Shiny.Locations background GPS and Spatial into one unified API</RN>
