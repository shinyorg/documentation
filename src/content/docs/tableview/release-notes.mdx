## 1.0.2 - February 24, 2026
<RN type="fix" platform="iOS">Section header and footer text is now readable in dark mode using iOS secondary label colors</RN>
<RN type="enhancement">EntryCell now uses a clean named Style (transparent background, no border) instead of inline property settings</RN>

## 1.0.1 - February 19, 2026
<RN type="fix" platform="iOS">DatePickerCell and TimePickerCell now use Focus() on iOS/Mac instead of transparent overlay, fixing tap interaction</RN>
<RN type="enhancement">Section header background now always applies a theme-aware default (light: #F2F2F7, dark: #1C1C1E) instead of only when explicitly set</RN>
<RN type="enhancement">Separator colors now use iOS system separator colors by default instead of low-opacity BoxView</RN>
<RN type="enhancement">Drag-sort reordering replaced up/down arrow buttons with native drag-and-drop gesture recognizers</RN>

## 1.0 - February 21, 2026
<RN type="feature">Initial Public Release</RN>
<RN type="feature">14 built-in cell types: LabelCell, CommandCell, ButtonCell, SwitchCell, CheckboxCell, SimpleCheckCell, RadioCell, EntryCell, DatePickerCell, TimePickerCell, TextPickerCell, NumberPickerCell, PickerCell, and CustomCell</RN>
<RN type="feature">Three-level cascading style system (TableView, Section, Cell)</RN>
<RN type="feature">Dynamic sections and cells via ItemsSource / ItemTemplate with INotifyCollectionChanged support</RN>
<RN type="feature">Drag and sort reordering within sections</RN>
<RN type="feature">RadioCell with section-scoped or TableView-scoped radio groups</RN>
<RN type="feature">PickerCell with full-page single or multi-select and auto-generated value text</RN>
<RN type="feature">Pure .NET MAUI implementation with zero platform-specific code</RN>
<RN type="feature">Full MVVM support with BindableProperty on all properties</RN>
<RN type="feature">Programmatic scroll control</RN>
<RN type="feature">Section headers and footers with text or custom views</RN>
<RN type="feature">Cell icons, hints, descriptions, and borders</RN>
<RN type="feature">Selection highlighting with customizable colors</RN>
