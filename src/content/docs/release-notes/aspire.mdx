<RN type="feature" repo="aspire">Orleans silo ADO.NET configuration via `UseOrleansWithAdoNet`</RN>

<RN type="feature" repo="aspire">Feature selection flags for schema provisioning (Clustering, Persistence, Reminders)</RN>
